# To-do-app
![Screenshot here](/Todoapp.png)

## How to use:
1. Write something in the input text (an alert will appear if there is nothing in the input text field)
2. Press the "Enter" key and the thing to do will appear on a list
3. Click on the item in the list to check it
4. Press the "X" button next to the item to delete it
5. Press the "Clear all" button to delete all items
6. What you wrote will be saved to avoid having to write everything again

Try it out: [To do App](https://selectric28.github.io/To-do-app/)
